Paper installation



Spigot/Paper/Purpur/etc. have a bug that basically causes world generation packs to not work, with regular worldgen being applied instead.
FSEE is no exception for this one...

How to prevent/fix this?

1. Drop the datapack into its directory (world/datapacks)
2. Turn on your server
3. Wait until it finishes loading, and the turn it off
4. Head to world_the_end folder and delete DIM1 folder found inside
5. Start the server

Works 99% of the time
The other 1% is you being unable to do these steps correctly.
