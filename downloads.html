Server Installation



This section applies only to MC versions 1.19.3 and prior!!

Minecraft servers have a bug which causes worldgen packs to not load initially, causing server to apply default worldgen settings on startup. This result in vanilla world generation, ignoring datapack configs.


How to prevent/fix this?

1️⃣ Drop the datapack into its directory (world/datapacks)
2️⃣ Start your server
3️⃣ Wait until it finishes loading, and then stop it
4️⃣ Head to 'world_the_end' folder and delete 'region' and 'entities' folders found inside
Or 'world/DIM1' on Fabric, or straight to 'DIM1' on Vanilla servers
5️⃣ Start your server again

Works 99% of the time!
If you happen to be that unfortunate 1%, just try again!
'Now Until The Moment It Works!'
